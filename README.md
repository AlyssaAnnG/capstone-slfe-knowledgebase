# SFLE Knowledgebase

## Running the Project

**Get Started**

1. Clone or download this repository
2. `$ npm install` on the directory

**Development Mode**

1. `$ npm start`
3. Browsersync can be accessed through `localhost:5000`

**Production Mode**
1. `npm build`

**Running Server**
1. `MONGODB_URI=[mongodbURI] npm run server`
